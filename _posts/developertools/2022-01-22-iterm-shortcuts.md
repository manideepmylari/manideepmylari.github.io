# Terminal Productivity - iTerm-2

| **Shortcut** | **For**                    |
|:-------------|:---------------------------|
| **⌘ ,**      | Open Preferences           |
| **⌘ D**      | Split Windows Vertically   |
| **⌘ ⇧ D**    | Split Windows Horizontally |
| **⌘ ;**      | Auto completion            |
| **⌘ ⇧ H**    | Paste History              |
| **⌘ ⌥ B**    | Show Interactive History   |
| **⌥ ⌘ Drag** | Select Rectangular Block   |
