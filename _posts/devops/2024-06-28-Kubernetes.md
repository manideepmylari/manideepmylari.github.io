# Minikube
[https://minikube.sigs.k8s.io/docs/handbook/controls/](https://minikube.sigs.k8s.io/docs/handbook/controls/)
```shell
brew install minikube
```

Start/shutdown the cluster
```shell
minikube start
minikube start
```

Dashboard
```shell
minikube dashboard
```
